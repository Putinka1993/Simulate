# Проект: Интеграция с API и загрузка данных в PostgreSQL и Google Sheets

## Описание проекта
Скрипт для получения данных об использовании редактора кода студентами онлайн-университета, их обработки и загрузки в локальную базу данных PostgreSQL и Google Sheets.

## Состав проекта

### 1. Получение данных из API
- Отправка GET-запроса к API с параметрами client, client_key, start, end
- Получение данных в формате JSON
- Преобразование строки passback_params в словарь
- Извлечение необходимых полей
- Формирование датафрейма df_selected

### 2. Загрузка данных в PostgreSQL
- Создание таблицы student_attempts
- Валидация данных перед вставкой
- Пакетная загрузка с коммитами каждые 100 записей
- Обработка ошибок при вставке

### 3. Логирование
- Запись логов в папку logs
- Формат имени файла: ГГГГ-ММ-ДД.log
- Автоматическое удаление логов старше 3 дней
- Фиксация всех этапов работы и ошибок

### 4. Агрегация данных
Расчет следующих метрик:
- Всего попыток
- Успешных попыток
- Уникальных пользователей
- Процент успеха
- Количество попыток по типам (run/submit)
- Период данных (дата начала и окончания)

### 5. Загрузка в Google Sheets
- Авторизация через сервисный аккаунт
- Создание или очистка листа Summary
- Загрузка агрегированных данных
- Добавление временной метки last_updated
