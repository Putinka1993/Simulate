## 1. Аналитическое исследование (RFM + ABC)

Файл: **RFM+ABC_pharmacy.ipynb**

**Цель исследования**:  
Сезонный спад выручки в аптеке приближается. Нужно провести персонализированные SMS-рассылки, чтобы не тратить бюджет впустую.

**Задачи**:
- Классифицировать клиентов с помощью RFM-анализа
- Дополнительно провести ABC-анализ товаров
- Сформировать рекомендации: какой сегмент клиентов получает какое предложение

**Ключевые элементы**:
- RFM-метрики (Recency, Frequency, Monetary) с тертилями
- 27 сегментов клиентов (от 111 до 333)
- Рекомендации по SMS и акциям для разных групп (Чемпионы, Спящие киты, В зоне риска и т.д.)
